# Screens & Shades Website

A professional website for a screens and shades business featuring motorized, manual, and pet mesh solutions.

## What's Included

- **index.html** - Main landing page with all sections
- **styles.css** - Modern, responsive styling
- **README.md** - This file with instructions

## Sections

1. **Navigation** - Easy access to all sections
2. **Hero Section** - Eye-catching welcome area
3. **Products** - Showcase of your offerings (motorized, manual, pet mesh, shades, commercial, custom)
4. **About** - Information about your business and services
5. **Contact** - Contact form and business information
6. **Footer** - Copyright information

## How to Customize

Edit the files to add your own content:

1. **Company Name**: Change "Screens & Shades Co." throughout the HTML
2. **Contact Information**: Update phone, email, address, and hours in the Contact section
3. **Product Descriptions**: Modify the product card text
4. **Colors**: Edit the color values in `styles.css` (look for `:root` section)
5. **About Section**: Update your company description
